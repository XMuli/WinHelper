## Spy++

::: tip
按一次快捷鍵開始，再按一次結束。若目標視窗需要系統管理員許可，請以系統管理員身份執行以獲取。
:::

## 關閉 Microsoft Defender

一鍵關閉 Microsoft Defender 防毒。

::: tip
**此功能需系統管理員權限**，少數情況可能無法立即生效（視窗未刷新），可嘗試重啟系統後再確認。
:::
