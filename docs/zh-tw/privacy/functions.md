## Defender 開關
快速開啟或關閉 Microsoft Defender 防毒（需系統管理員）。  
**模式**  
- 停止：Defender 關閉且不掃描。  
- 啟動：由 Windows 管理；若有其他防毒，Defender 會被停用。  
- 強制啟用：無論是否有其他防毒，Defender 都會運行。
